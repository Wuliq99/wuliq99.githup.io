  • 移位符右边的数一定要是整数，下列例子会编译错误

  • C语言里函数的参数默认是从右往左处理的，输出时是从左往右。

    • #include<stdio.h>
int main(void)
{
int a = 10, b = 20, c = 30;
 printf("\n %d..%d..%d \n", a+b+c, (b =b*2),(c = c*2));
 return 0;
 }

      结果：110..40..60

  • C语言中，非0为真，零为假

  • 在Dev C++ 中，指针的大小都为 8 个字节

  • sizeof ()内部的表达式不参与真实运算！这点很重要！其运算值在编译时就计算好了

  • 关于void*型指针：

    void*：无具体类型的指针 能够接收任意类型的地址
    缺点：不能进行运算。不能+-整数，不能解引用

  • sizeof()与strlen()

共同点：返回类型都是size_t

不同点：sizeof()是操作符计算的是变量/类型所占空间的大小，单位是字节，\0也算进空间

​ strlen()是库函数，计算的是字符串长度，不计算\0

  • 在两种方法都正确时，下标绝不会比指针更有效率，但指针有时会比下标更有效率。前提是指针被正确使用
  • 下标引用的优先级高于间接访问
